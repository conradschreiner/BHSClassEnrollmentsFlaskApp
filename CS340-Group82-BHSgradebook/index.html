Description of Webpages

GradeLevels
GradeLevel Table Directory. This webpage presents the current GradeLevel records in the database, with the DDL file loading the default Freshman, Sophomore, Junior, and Senior.
Students
Lists all records in the Students table in the BHS database and new students may be added to the table by the user here. There will be future functionality for the user to remove a student record from the database here via a SQL DELETE statement.
Teachers
Lists all records in the Teachers table in the BHS database and new teachers may be added to the table by the user here.
Departments
Lists all records in the Departments table and new departments may be added to the table by the user here.
Courses
Lists all records in the Courses table and new courses may be added to the table by the user here.
ClassSections
Lists all records in the ClassSections table and new class sections may be added to the table by the user here.
Enrollments
Lists all records in the Enrollments table and new enrollments may be added to the table by the user here.
